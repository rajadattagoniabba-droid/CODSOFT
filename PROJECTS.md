# CODSOFT Projects

This repository contains multiple mini-projects developed during the CODSOFT internship.

## Included Projects
- Calculator
- E-Commerce Website
- Job Board
- Landing Page
- Myportfolio
- Online Quiz Maker
- Project Management Tool

## Structure
Each project folder contains its own frontend/backend files and supporting assets.

## Getting Started
1. Open any project folder.
2. Install dependencies if required.
3. Run the project using the provided launcher or package scripts.

## Notes
- Some projects are static HTML/CSS/JS pages.
- Others are Vite-based React apps with Node.js backends.
