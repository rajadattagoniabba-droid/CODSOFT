# CODSOFT

This repository contains multiple mini-projects completed during the CODSOFT web development internship.

## Projects Included
- Calculator
- E-Commerce Website
- Job Board
- Landing Page
- Myportfolio
- Online Quiz Maker
- Project Management Tool

## Tech Stack
These projects use a mix of HTML, CSS, JavaScript, React, Vite, Node.js, and Express.

## How to Use
1. Open any project folder.
2. Install dependencies if required.
3. Run the app using the provided launcher or package scripts.

## Notes
This repository is prepared as a single GitHub repository with shared project-level ignore rules and a unified structure.
